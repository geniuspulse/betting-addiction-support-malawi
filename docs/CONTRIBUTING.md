# Contributing to BASM

Thank you for your interest in contributing to Betting Addiction Support Malawi!

## How to Contribute
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add some feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

## Code of Conduct
Be respectful. This project serves vulnerable people — maintain sensitivity in all contributions.

## Reporting Issues
Open a GitHub Issue with clear steps to reproduce any bug.
